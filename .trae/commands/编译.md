## 编译步骤

1. 执行编译命令（使用 MSBuild）：
```powershell
& "D:\VS\MSBuild\Current\Bin\MSBuild.exe" "d:\opencpn-plugin-template-main\build\ncdf_pi.vcxproj" /p:Configuration=Release /p:Platform=Win32 /m /v:minimal
```

2. 检查编译结果：
```powershell
Get-Item d:\opencpn-plugin-template-main\build\Release\ncdf_pi.dll | Select-Object Name, LastWriteTime
```
- 确认 DLL 文件的生成时间与当前时间接近
- 若时间不符说明编译未成功，需删除缓存文件重新编译

## 部署步骤

编译完成后，将新编译的 DLL 复制到 OpenCPN 插件目录：

```powershell
Copy-Item d:\opencpn-plugin-template-main\build\Release\ncdf_pi.dll "C:\Users\季曹阳\AppData\Local\opencpn\plugins\" -Force
```

## 完整命令脚本

```powershell
# 编译插件

Write-Host "`n=== 检查编译结果 ==="

# 部署到插件目录
Write-Host "`n=== 部署插件 ==="

```
